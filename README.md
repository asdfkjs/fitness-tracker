# 💪 健身打卡日历 (Fitness Checkin Calendar)

一个类似 GitHub 贡献日历风格的健身打卡工具，纯前端实现，数据保存在浏览器本地。

## ✨ 功能特性

- **GitHub 贡献日历风格**：以周为行（周一到周日），构成一整年的热力图网格
- **年份切换**：可查看不同年份的打卡记录，默认显示当前年份
- **打卡记录**：点击格子弹出模态框，输入当天的健身内容
- **数据统计**：当前连续打卡天数、总打卡天数、本年度打卡天数
- **数据持久化**：使用 localStorage 保存，刷新页面不丢失
- **移动端适配**：日历区域可横向滚动，模态框自适应小屏
- **零依赖**：单个 HTML 文件，无需任何外部资源

## 🚀 使用方法

1. 直接在浏览器中打开 `fitness-tracker.html`
2. 点击任意不晚于今天的格子
3. 在弹出的模态框中输入健身内容
4. 点击"保存"完成打卡

## 📦 部署方式

### 方式一：直接打开
双击 `fitness-tracker.html` 文件即可在浏览器中使用。

### 方式二：部署到 GitHub Pages
1. Fork 或 Clone 本仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 访问 `https://你的用户名.github.io/仓库名/`

### 方式三：嵌入 WordPress
1. 打开 WordPress 后台
2. 新建/编辑页面，添加"自定义 HTML"区块
3. 将 `fitness-tracker.html` 的全部内容粘贴进去
4. 发布页面
